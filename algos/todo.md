# Алгоритмы для диплома

1. Алгоритм вытеснения LRU (Least Recently Used)
2. Алгоритм консистентного хеширования (Consistent Hashing)
3. Алгоритм распространения информации Gossip (Epidemic Dissemination)
4. Алгоритм Leader Election через Lease в Kubernetes
5. Алгоритм параллельного скачивания чанков с выбором оптимального пира (Peer Selection)
6. Алгоритм Prefetching (упреждающее чтение) на основе последовательного паттерна доступа
