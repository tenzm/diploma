Объектом разработки в данной работе является алгоритм и конвейер по редактированию виде данных при помощи текстовых запросов. 

Цель работы – создание алгоритма машинного обучения по текстовому редактированию видеоданных и максимально приблизить данный алгоритм к работе в реальном времени. 

Для достижения поставленной цели были проведены исследования в области методов ускорения работы нейросетевых моделей, технологий создания больших языковых моделей и диффузионных моделей.
